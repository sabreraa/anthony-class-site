Website is a Shifting House

I relate to the students that ask themselves questions about how to make a website because, to be honest, it can be anything. For me, it helps to have a definitive definition, topic, theme, or direction when it comes to a website, but after reflecting on this reading, I learn that it can be anything you create. It is very flexible, where you can add as many files and whatever file of the information you want. It is similar to an ecosystem, where files make up the website.
